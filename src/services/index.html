> LLDSとはこちら

最短3日で導入可能

お手持ちのFAQデータを無加工で取り込むことができるので、導入に手間がかかりません。

外部サービス連携

Microsoft Teams・Microsoft Azure・Skype等のAPIと連携可能なので、社内グループウェアにもご利用頂けます。

LLDS（Log Learning Dictionary System）は
ARアドバンストテクノロジの保有するオリジナル辞書データをベースとし、
お客様のFAQデータやFAQログによる強化辞書を自動的に生成するためのシステムです。

Features

機能紹介

自動学習

FAQの質問と回答をすべて記録し、形態素解析でワード分解。既存ワードとの関係性を自動更新します。初期導入および日々のFAQメンテナンスの負荷を軽減します。

聞き返し機能

お客様の質問に対して聞き返しを行い、お客様の本当に聞きたい疑問に対する答えに導きます。

チャット・SNS連携

スマートフォンアプリ、グループウェア、チャットアプリ、ロボット等、LOOGUEのREST APIを呼び出すことのできるプラットフォームと連携可能です。

付帯情報取り込み

お客様がお持ちのマニュアルや製品情報を読み込み、FAQの内容に連動して表示させることができます。

キャラクター設定

アニメーション付きのキャラクターを設定でき、ユーザ様に親近感を持って頂けます。

サジェスト対応

検索した文字列に関連の深い語句を逐次予測して表示することができる機能です。

System coordination

機能連携例

※利用者となるお客様が企業の場合、企業内でOffice365のTeamsのようなSNSツールを使ってコミュニケーションを行っているのであれば、SNSツールを経由してLOOGUEのFAQを利用することが可能です。

※お客様がご利用しているSNSツールからLOOGUEに接続できる環境をご用意いただく必要があります。

※コールセンター様が利用者向けに提供するチャット管理システムの１人のオペレータとしてLOOGUEを連携させることが可能です。

※コールセンター様のチャット管理システムからLOOGUEサーバに接続できる環境をご用意いただく必要があります。またチャット管理システム製品の仕様に応じ、インターフェースを開発する必要があります。
